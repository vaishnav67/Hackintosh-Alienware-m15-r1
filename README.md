* CPU: Intel Core i7-8750H
* iGPU: Intel UHD Graphics 630
* dGPU: NVIDIA GeForce RTX 2060
* WiFi Chip: Killer Wireless AC-1550 (9260NGW)
* Ethernet: Killer E2500
